# 期末课程设计:图书销售管理系统――图书贤者

### 技术栈选择:
- ASP.Net:Blazor前后端不分离,无需编写JavaScript,可以让我专注于页面的逻辑
- SQLite:轻量级数据库,无需额外安装数据库程序
- EF core:选用的ORM框架,无需手动编写SQL语句,增强程序安全性,便于开发
